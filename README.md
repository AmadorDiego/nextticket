# nextticket

Monorepo con arquitectura SOFEA y aplicaciones separadas en frontend, backend y mobile.

## Estructura principal

- `agents/`
- `apps/`
- `docs/`
- `infra/`
- `packages/`
- `scripts/`
- `stubs/`

## Comandos raíz

- `pnpm install`
- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm test`
